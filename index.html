Build Solutions

With integration with the top Ethereum clients to blockchain services and clouds, Nethereum helps you build blockchain solutions.

Nethereum Blockchain Store library for Azure

Sometimes indexing blockchain data enables more efficient query and trigger processing. This Nethereum library uses Azure Table Storage, a simple (and cheap) way to store data that can be easily replaced with other implementations like Azure SQL. Github

Ethereum Flappy Unicorn in Unity3d

Nethereum includes support for Unity3d, one of the most widely used game engines with development support for Xbox, PS4, iOS, Android, UWP, Hololens, SamsungTV, WegGL, WebAssembly. Using Urho3d by Xamarin a 3D mobile game was also created.

Play Online, read the Blog Post, view the Sourcecode, or see the mobile game

Nethereum-Blazor-WASM

This was an experimental dapp (decentralized app) using Blazor, .NET Web Assembly and Metamask. This enables a .NET application running in the browser. Learn more on Github.

Nethereum.UI.Wallet.Sample

A cross platform wallet example using Nethereum, Xamarin.Forms and MvvmCross, targetting all main mobile platforms Android, iOS, Windows, Dektop (Windows 10 uwp), IoT with Raspberry PI and Xbox. Github
